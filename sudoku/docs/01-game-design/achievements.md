- ปลดล็อกแล้วได้ **coin + XP + บางอันได้ exclusive item**

## 🗄️ Storage

```sql
CREATE TABLE achievements_definitions (
  id TEXT PRIMARY KEY,
  name TEXT NOT NULL,
  description TEXT NOT NULL,
  tier TEXT NOT NULL,
  reward_coin INTEGER DEFAULT 0,
  reward_xp INTEGER DEFAULT 0,
  reward_item_id TEXT,  -- nullable
  category TEXT NOT NULL,
  hidden BOOLEAN DEFAULT false,  -- ซ่อนไว้ให้เซอร์ไพรส์
  icon TEXT
);

CREATE TABLE user_achievements (
  user_id UUID NOT NULL,
  achievement_id TEXT NOT NULL,
  unlocked_at TIMESTAMPTZ NOT NULL DEFAULT now(),
  PRIMARY KEY (user_id, achievement_id)
);
```

## 📐 Trigger Implementation

ทุกครั้งที่ user complete game / get reward → run **achievement checker**:

```ts
async function checkAchievements(userId: string, event: GameEvent) {
  const userStats = await getUserStats(userId);
  const unlocked = await getUserAchievements(userId);

  for (const ach of ACHIEVEMENT_DEFINITIONS) {
    if (unlocked.has(ach.id)) continue;
    if (await checkCondition(ach, userStats, event)) {
      await unlockAchievement(userId, ach.id);
      await grantRewards(userId, ach.rewards);
      notify(userId, ach);
    }
  }
}
```

**Performance:** เก็บ `userStats` cached + only recompute relevant on each event.

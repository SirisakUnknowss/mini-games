# Sudoku Daily v0.1.0

**Release date:** 2026-05-31
**Build:** versionCode 1 · versionName 1.0
**Platform:** Android (signed AAB + APK) · iOS (Xcode Archive)

---

## ✨ What's new

### Phase 1 — MVP
- 📅 Daily puzzle fetched from Supabase — new puzzle every midnight UTC
- 🔐 Auth: sign up / sign in / anonymous guest mode / upgrade to email
- 🏆 Live leaderboard — Today / Yesterday tabs with Supabase Realtime
- 🎯 Daily quests — progress bars + claim rewards
- 🔥 Streak system — milestone toasts at 3/7/14/30/60/100/365 days
- 🧭 4-step onboarding wizard (shown once)

### Phase 2 — Customization
- 🎨 11 themes (Classic, Paper, Dark, Pastel, Ocean, Forest, Sunset, Neon, Sakura, Thai, Mono Pro)
- 🖼 9 backgrounds incl. 3 animated (Rain, Stars, Aurora)
- 🛍 Shop: Buy / Equip / Preview-on-hover
- 👤 Avatar picker (21 emoji) + 13 shop avatar items with distinct icons
- ✨ Coin count-up animation

### Phase 3 — Progression
- ⭐ XP / Level system + in-home XP bar + level-up modal
- 🏆 53 achievements with progress bars
- 📊 Stats dashboard + time-per-game chart + vs Global average
- 📅 Weekly Recap + share
- 💰 Coin Ledger view

### Bug fixes
- Nav buttons on Leaderboard / Shop / Profile were not working (wireBottomNav fix)
- Sign out button was showing for guests
- Avatar shop items all showed the same 👤 icon
- Stats / Recap queries used wrong column names (`played_at` → `completed_at`)

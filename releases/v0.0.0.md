# v0.0.0 — Sudoku Daily MVP Foundation

> Phase 0 complete. Engine + infrastructure + mobile builds working.

This is the **foundation release** of Sudoku Daily — a Sudoku game designed for daily engagement with global leaderboard, customization, and progression.

**The other games in this repo** (`match-three/`, `suika/`) remain as v1 vanilla JavaScript and are not part of this release.

---

## 🧩 What's Working

### Web (PWA)
- Full Sudoku gameplay (daily + practice modes, 4 difficulties)
- Animated splash screen
- 4 ranked levels (Easy/Medium/Hard/Expert) + Daily Puzzle
- Touch + keyboard input
- Highlight options (same number, row/col/box, conflicts)
- Hint system (3 per game)
- Score + timer + mistake counter
- Share results (Wordle-style)
- Installable as PWA

### Mobile (Capacitor 8)
- Android APK (5.6 MB) — attached to this release
- iOS app builds (Simulator + device-ready)
- Custom branded icons (100+ assets generated)
- **Adaptive icon with monochrome layer** for Android 13+ themed icons
- Animated splash with logo sequence

## 🏗️ Architecture (TypeScript + Vite)

```
src/
├── engine/     Pure logic — generator, solver (with uniqueness check),
│               validator, scoring, quests, RNG
├── lib/        Supabase, auth, API, analytics, IndexedDB, v1 migration
├── ui/         Views, components, styles (with theme tokens)
└── state/      Zustand store
```

**Quality:**
- ✅ 37/37 unit tests passing (vitest)
- ✅ TypeScript strict mode
- ✅ ~65 KB initial JS gzipped
- ✅ Self-check script: `npm run phase0:verify` (11 sections)

---

## 🗄️ Backend (Supabase, ready to deploy)

- 5 SQL migrations (schema, RLS, triggers, views, RPC functions)
- Seed data: 30+ shop items, 19 achievements
- Edge function: `submit-daily-score` with anti-cheat (move replay, sanity checks, server-side scoring)
- RPC functions: `grant_coins`, `grant_xp`, `bump_streak_for_today`

To deploy:
```bash
cd sudoku
supabase link --project-ref <YOUR_PROJECT_REF>
supabase db push
supabase functions deploy
```

## 🔄 v1 Migration

The v1 of Sudoku (vanilla JS in `sudoku/` before this release) lived in localStorage. v2 includes:
- Idempotent boot-time migration
- Imports: settings, practice progress (best per stage), recent history
- Bonus: 200 starter coins for migrators
- Audit trail in `coin_transactions`

## 📱 Try It

### Web
1. Clone the repo
2. `cd sudoku && npm install`
3. `npm run dev`
4. Open http://localhost:5173

### Android
Download `sudoku-daily-v0.0.0.apk` below, transfer to phone, install (allow "Unknown sources").

### iOS
Build from source:
```bash
cd sudoku
npm install && npm run build
npx cap sync ios
npx cap open ios
# Then in Xcode: configure Team → Build → Run on simulator/device
```
